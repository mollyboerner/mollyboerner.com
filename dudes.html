Dudes

For this project, we were told to use patterns for basic 3-Dimensional shapes and find new ways to change them. The goal was to learn how new shapes can be created from old ones. For my shapes, I chose cylinders and cones.

The fuzzy man started as a cone, but I gathered and stitched one side, creating a curved,ruched edge. After looking at it, I though it resembled a small creature so I added some eyes and some fur.

The snake/worm started as a bunch of cylinders sewn together. Using the same ruching method, I made them curl. I wanted to see the extent to which I could make them curve and figured using a longer shape would help.

My final piece was a cylinder. While I had no particular method I wanted to test out on the cylinder, I though it could be fun to create a more interactive piece. After making the cylinder, I did a loose basting stitch around certain parts of the cylinder. This allowed the user to pull on whichever strings they wanted to create a new shape.
